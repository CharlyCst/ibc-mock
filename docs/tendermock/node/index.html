[−][src]Module tendermock::node

Node

A tendermock node, which encapsulate a storage and a chain. This also exposes a thread safe version called SharedNode for use by the various RPC interfaces.

To integrate with IBC modules, the node implements the ICS26Context traits, which mainly deal with interacting with storing and reading values.
